# Passepartout

A demo app for [Latchkey](https://github.com/imbue-ai/latchkey).

## Tech Stack

- **Frontend**: React + TypeScript
- **Backend**: Rust (Tauri)
- **Build Tool**: Vite

## Prerequisites

- Node.js (see `.nvmrc` for version)
- Rust (latest stable)
- Tauri CLI

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run tauri dev
```

## Build

Build for production:

```bash
npm run tauri build
```
